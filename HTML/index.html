Projeto monitoramento de servidor WEB

Ver no GitHub

Este é um projeto em ShellScript desenvolvido para monitorar um servidor web e enviar uma notificação no discord caso ocorra alguma falha no sistema

Objetivo

O objetivo principal

  • Escolher uma distribuição Linux
  • Lançar uma EC2
  • Criar um WEBHOOK para discord
  • Instalar o Nginx
  • Criar um script de monitoramento para o servidor web

Tecnologias Utilizadas

ShellScript (Bash)

Funcionalidades Principais

  1. Monitorar o servidor web
  2. Script executado de 1 em 1 minuto
